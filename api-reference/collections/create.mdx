## Create Collection

Create a new collection to organize and store knowledge bases, documents, and other content with vector embeddings.

### Headers

<ParamField header="Authorization" type="string">
  Bearer token for authentication
</ParamField>

<ParamField header="X-API-Key" type="string">
  API key for authentication (alternative to Authorization header)
</ParamField>

<ParamField header="X-Workspace-Id" type="string">
  Optional. Derived from API Key, but can be specified if you have access to multiple workspaces
</ParamField>

### Request Body

<ParamField body="name" type="string" required>
  Collection name (1-100 characters)
</ParamField>

<ParamField body="description" type="string">
  Collection description (up to 255 characters)
</ParamField>

<ParamField body="store_type" type="string">
  Storage type for the collection (pgvector, chroma, pinecone, milvus, qdrant, mongodb, upstash)
</ParamField>

<ParamField body="visibility" type="string" default="private">
  Collection visibility (private, public)
</ParamField>

<ParamField body="status" type="string" default="active">
  Collection status (active, archived)
</ParamField>

<ParamField body="tags" type="array">
  Collection tags for organization
</ParamField>

<ParamField body="embedding" type="object">
  Embedding configuration
  
  <Expandable title="embedding object">
    <ParamField body="provider" type="string">
      Embedding provider (openai, azure_openai, huggingface, etc.)
    </ParamField>
    
    <ParamField body="model" type="string">
      Embedding model name
    </ParamField>
    
    <ParamField body="dimensions" type="integer">
      Embedding dimensions
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="meta_data" type="object">
  Additional metadata for the collection
</ParamField>

<ParamField body="settings" type="object">
  Collection configuration settings
</ParamField>

### Response

<ResponseField name="id" type="string">
  Unique collection identifier
</ResponseField>

<ResponseField name="name" type="string">
  Collection name
</ResponseField>

<ResponseField name="description" type="string">
  Collection description
</ResponseField>

<ResponseField name="store_type" type="string">
  Storage type
</ResponseField>

<ResponseField name="visibility" type="string">
  Collection visibility
</ResponseField>

<ResponseField name="status" type="string">
  Collection status
</ResponseField>

<ResponseField name="tags" type="array">
  Collection tags
</ResponseField>

<ResponseField name="embedding" type="object">
  Embedding configuration
</ResponseField>

<ResponseField name="workspace_id" type="string">
  Workspace ID this collection belongs to
</ResponseField>

<ResponseField name="created_by" type="string">
  User who created the collection
</ResponseField>

<ResponseField name="created_at" type="string">
  Creation timestamp
</ResponseField>

```python Python
import requests

response = requests.post(
    "https://api.mielto.com/api/v1/collections",
    headers={
        "X-API-Key": "YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "name": "Product Documentation",
        "description": "All product documentation and guides",
        "store_type": "pgvector",
        "visibility": "private",
        "tags": ["docs", "product"],
        "embedding": {
            "provider": "openai",
            "model": "text-embedding-ada-002",
            "dimensions": 1536
        },
        "meta_data": {
            "category": "documentation"
        }
    }
)

collection = response.json()
```
