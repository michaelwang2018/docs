## Archive Collection

Archive a collection to preserve its content while removing it from active use. Archived collections are not searchable but can be restored later.

### Headers

<ParamField header="Authorization" type="string">
  Bearer token for authentication
</ParamField>

<ParamField header="X-API-Key" type="string">
  API key for authentication (alternative to Authorization header)
</ParamField>

<ParamField header="X-Workspace-Id" type="string">
  Optional. Derived from API Key, but can be specified if you have access to multiple workspaces
</ParamField>

### Path Parameters

<ParamField path="collection_id" type="string" required>
  The unique identifier of the collection to archive
</ParamField>

### Response

Returns the archived collection object with updated status.

<RequestExample>

```bash cURL
curl -X POST "https://api.mielto.com/api/v1/collections/col_123456789/archive" \
  -H "X-API-Key: YOUR_API_KEY"
```

```python Python
import requests

collection_id = "col_123456789"
response = requests.post(
    f"https://api.mielto.com/api/v1/collections/{collection_id}/archive",
    headers={"X-API-Key": "YOUR_API_KEY"}
)

archived_collection = response.json()
```

```javascript JavaScript
const collectionId = "col_123456789";
const response = await fetch(`https://api.mielto.com/api/v1/collections/${collectionId}/archive`, {
  method: "POST",
  headers: {
    "X-API-Key": "YOUR_API_KEY"
  }
});

const archivedCollection = await response.json();
```

</RequestExample>
