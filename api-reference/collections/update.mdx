## Update Collection

Update an existing collection's name, description, settings, or other properties.

### Headers

<ParamField header="Authorization" type="string">
  Bearer token for authentication
</ParamField>

<ParamField header="X-API-Key" type="string">
  API key for authentication (alternative to Authorization header)
</ParamField>

<ParamField header="X-Workspace-Id" type="string">
  Optional. Derived from API Key, but can be specified if you have access to multiple workspaces
</ParamField>

### Path Parameters

<ParamField path="collection_id" type="string" required>
  The unique identifier of the collection to update
</ParamField>

### Request Body

<ParamField body="name" type="string">
  Updated collection name (1-100 characters)
</ParamField>

<ParamField body="description" type="string">
  Updated collection description (up to 255 characters)
</ParamField>

<ParamField body="visibility" type="string">
  Updated collection visibility (private, public)
</ParamField>

<ParamField body="status" type="string">
  Updated collection status (active, archived)
</ParamField>

<ParamField body="tags" type="array">
  Updated collection tags
</ParamField>

<ParamField body="meta_data" type="object">
  Updated metadata for the collection
</ParamField>

<ParamField body="settings" type="object">
  Updated collection configuration settings
</ParamField>

<ParamField body="embedding" type="object">
  Updated embedding configuration
</ParamField>

### Response

Returns the updated collection object.

<RequestExample>

```bash cURL
curl -X PUT "https://api.mielto.com/api/v1/collections/col_123456789" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Updated Product Documentation",
    "description": "Updated description for product docs",
    "tags": ["docs", "product", "updated"],
    "meta_data": {
      "category": "documentation",
      "version": "2.0"
    }
  }'
```

```python Python
import requests

collection_id = "col_123456789"
response = requests.put(
    f"https://api.mielto.com/api/v1/collections/{collection_id}",
    headers={
        "X-API-Key": "YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "name": "Updated Product Documentation",
        "description": "Updated description for product docs",
        "tags": ["docs", "product", "updated"],
        "meta_data": {
            "category": "documentation",
            "version": "2.0"
        }
    }
)

updated_collection = response.json()
```
