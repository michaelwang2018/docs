## Get User Profile

Retrieve detailed profile information for the currently authenticated user.

### Headers

<ParamField header="X-API-Key" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField header="X-Workspace-Id" type="string">
  Optional workspace ID header
</ParamField>

### Response

<ResponseField name="id" type="string">
  Unique user identifier
</ResponseField>

<ResponseField name="first_name" type="string">
  User's first name
</ResponseField>

<ResponseField name="last_name" type="string">
  User's last name
</ResponseField>

<ResponseField name="email" type="string">
  User's email address
</ResponseField>

<ResponseField name="avatar_url" type="string">
  URL to user's avatar image
</ResponseField>

<ResponseField name="auth_strategy" type="object">
  Authentication strategy information
</ResponseField>

<ResponseField name="last_login_at" type="string">
  Timestamp of last login (ISO 8601 format)
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://api.mielto.com/api/v1/users/profile" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "X-Workspace-Id: wsp_123456"
```

```python Python
import requests

response = requests.get(
    "https://api.mielto.com/api/v1/users/profile",
    headers={
        "X-API-Key": "YOUR_API_KEY",
        "X-Workspace-Id": "wsp_123456"
    }
)

profile = response.json()
```

```javascript JavaScript
const response = await fetch("https://api.mielto.com/api/v1/users/profile", {
  headers: {
    "X-API-Key": "YOUR_API_KEY",
    "X-Workspace-Id": "wsp_123456"
  }
});

const profile = await response.json();
```

</RequestExample>
