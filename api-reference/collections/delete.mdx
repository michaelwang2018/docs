## Delete Collection

Permanently delete a collection and all its associated content, including documents, vectors, and metadata.

<Warning>
  This action cannot be undone. All documents, vectors, and associated data in this collection will be permanently deleted.
</Warning>

### Headers

<ParamField header="Authorization" type="string">
  Bearer token for authentication
</ParamField>

<ParamField header="X-API-Key" type="string">
  API key for authentication (alternative to Authorization header)
</ParamField>

<ParamField header="X-Workspace-Id" type="string">
  Optional. Derived from API Key, but can be specified if you have access to multiple workspaces
</ParamField>

### Path Parameters

<ParamField path="collection_id" type="string" required>
  The unique identifier of the collection to delete
</ParamField>

### Response

Returns `204 No Content` on successful deletion.

<RequestExample>

```bash cURL
curl -X DELETE "https://api.mielto.com/api/v1/collections/col_123456789" \
  -H "X-API-Key: YOUR_API_KEY"
```

```python Python
import requests

collection_id = "col_123456789"
response = requests.delete(
    f"https://api.mielto.com/api/v1/collections/{collection_id}",
    headers={"X-API-Key": "YOUR_API_KEY"}
)

# Returns 204 No Content on success
print(f"Status Code: {response.status_code}")
```

```javascript JavaScript
const collectionId = "col_123456789";
const response = await fetch(`https://api.mielto.com/api/v1/collections/${collectionId}`, {
  method: "DELETE",
  headers: {
    "X-API-Key": "YOUR_API_KEY"
  }
});

// Returns 204 No Content on success
console.log(`Status Code: ${response.status}`);
```

</RequestExample>

<ResponseExample>
