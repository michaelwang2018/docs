## Update API Key

Update an existing API key's label, permissions, expiration, or other settings.

### Headers

<ParamField header="Authorization" type="string">
  Bearer token for authentication
</ParamField>

<ParamField header="X-API-Key" type="string">
  API key for authentication (alternative to Authorization header)
</ParamField>

<ParamField header="X-Workspace-Id" type="string">
  Optional. Derived from API Key, but can be specified if you have access to multiple workspaces
</ParamField>

### Path Parameters

<ParamField path="api_key_id" type="string" required>
  The unique identifier of the API key to update
</ParamField>

### Request Body

<ParamField body="label" type="string">
  Updated label for the API key
</ParamField>

<ParamField body="status" type="string">
  Updated status (active, inactive)
</ParamField>

<ParamField body="expires_at" type="string">
  Updated expiration date (ISO 8601 format)
</ParamField>

<ParamField body="permissions" type="array">
  Updated list of permissions
</ParamField>

<ParamField body="ip_whitelist" type="array">
  Updated list of allowed IP addresses
</ParamField>

<ParamField body="collections" type="array">
  Updated list of accessible collection IDs
</ParamField>

### Response

Returns the updated API key object.

<RequestExample>

```bash cURL
curl -X PUT "https://api.mielto.com/api/v1/api-keys/key_123456789" \
  -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "label": "Updated Production Key",
    "status": "active",
    "permissions": ["read", "write", "admin"],
    "expires_at": "2026-12-31T23:59:59Z"
  }'
```

```python Python
import requests

api_key_id = "key_123456789"
response = requests.put(
    f"https://api.mielto.com/api/v1/api-keys/{api_key_id}",
    headers={
        "X-API-Key": "YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "label": "Updated Production Key",
        "status": "active",
        "permissions": ["read", "write", "admin"],
        "expires_at": "2026-12-31T23:59:59Z"
    }
)

updated_key = response.json()
```

```javascript JavaScript
const apiKeyId = "key_123456789";
const response = await fetch(`https://api.mielto.com/api/v1/api-keys/${apiKeyId}`, {
  method: "PUT",
  headers: {
    "X-API-Key": "YOUR_API_KEY",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    label: "Updated Production Key",
    status: "active",
    permissions: ["read", "write", "admin"],
    expires_at: "2026-12-31T23:59:59Z"
  })
});

const updatedKey = await response.json();
```
