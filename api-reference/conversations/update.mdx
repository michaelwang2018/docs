## Update Conversation

Update an existing conversation's title, summary, status, settings, or other metadata.

### Headers

<ParamField header="Authorization" type="string">
  Bearer token for authentication
</ParamField>

<ParamField header="X-API-Key" type="string">
  API key for authentication (alternative to Authorization header)
</ParamField>

<ParamField header="X-Workspace-Id" type="string">
  Optional. Derived from API Key, but can be specified if you have access to multiple workspaces
</ParamField>

### Path Parameters

<ParamField path="conversation_id" type="string" required>
  The unique identifier of the conversation to update
</ParamField>

### Request Body

<ParamField body="title" type="string">
  Updated conversation title (max 255 characters)
</ParamField>

<ParamField body="summary" type="string">
  Updated conversation summary
</ParamField>

<ParamField body="status" type="string">
  Updated conversation status (active, ended, archived)
</ParamField>

<ParamField body="is_public" type="boolean">
  Updated visibility setting for the conversation
</ParamField>

<ParamField body="collections" type="array">
  Updated array of collection IDs to associate with this conversation
</ParamField>

<ParamField body="meta_data" type="object">
  Updated metadata for the conversation
</ParamField>

<ParamField body="settings" type="object">
  Updated conversation-specific settings
</ParamField>

<ParamField body="ended_at" type="string">
  Timestamp when conversation was ended (ISO 8601 format)
</ParamField>

<ParamField body="user_id" type="string">
  Updated user ID associated with this conversation
</ParamField>

### Response

Returns the updated conversation object with all current values.

```python Python

conversation_id = "conv_123456789"
response = requests.put(
    f"https://api.mielto.com/api/v1/conversations/{conversation_id}",
    headers={
        "X-API-Key": "YOUR_API_KEY",
        "Content-Type": "application/json"
    },
    json={
        "title": "Q2 Product Strategy & Market Analysis",
        "summary": "Comprehensive discussion covering Q2 product strategy, competitive analysis, and customer feedback integration",
        "meta_data": {
            "category": "planning",
            "priority": "high",
            "department": "product",
            "tags": ["strategy", "q2", "planning", "market-analysis"]
        },
        "settings": {
            "auto_summarize": True,
            "context_window": 6000,
            "max_tokens": 3000
        }
    }
)

updated_conversation = response.json()
```
