## List Workspaces

Retrieve workspaces accessible to the current user with pagination support.

### Headers

<ParamField header="X-API-Key" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField header="X-Workspace-Id" type="string">
  Optional workspace ID header
</ParamField>

### Query Parameters

<ParamField query="skip" type="integer" default="0">
  Number of workspaces to skip for pagination
</ParamField>

<ParamField query="limit" type="integer" default="100">
  Maximum number of workspaces to return (1-1000)
</ParamField>

### Response

Returns an array of workspace objects accessible to the user.

<ResponseField name="id" type="string">
  Unique workspace identifier
</ResponseField>

<ResponseField name="name" type="string">
  Workspace name
</ResponseField>

<ResponseField name="slug" type="string">
  URL-friendly workspace identifier
</ResponseField>

<ResponseField name="description" type="string">
  Workspace description
</ResponseField>

<ResponseField name="status" type="string">
  Workspace status (active, inactive, suspended)
</ResponseField>

<ResponseField name="logo_url" type="string">
  URL to workspace logo
</ResponseField>

<ResponseField name="created_at" type="string">
  Workspace creation timestamp
</ResponseField>

<ResponseField name="updated_at" type="string">
  Last update timestamp
</ResponseField>

<ResponseField name="created_by" type="string">
  ID of user who created the workspace
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://api.mielto.com/api/v1/workspaces?skip=0&limit=50" \
  -H "X-API-Key: YOUR_API_KEY"
```

```python Python
import requests

response = requests.get(
    "https://api.mielto.com/api/v1/workspaces",
    headers={"X-API-Key": "YOUR_API_KEY"},
    params={"skip": 0, "limit": 50}
)

workspaces = response.json()
```

```javascript JavaScript
const response = await fetch("https://api.mielto.com/api/v1/workspaces?skip=0&limit=50", {
  headers: {
    "X-API-Key": "YOUR_API_KEY"
  }
});

const workspaces = await response.json();
```
